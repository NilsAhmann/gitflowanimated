An experimental React app to visualize and animate Gitflow.

## Requirements

- Node.js 24

## Development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

Try it here: [https://veerasundar.com/blog/gitflow-animated](https://veerasundar.com/blog/gitflow-animated)
